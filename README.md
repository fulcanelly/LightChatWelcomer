# Welcomer_plugin

 
simple minecraft plugin for on join items and messages 

##config example

```yml

# config.yml
first-join-message: "Welcome to the server! %player%"
welcome-back-message: "Welcome back to the server! You were last seen on %lastseen%"
first-join-items:
  - DIAMOND 3
  - IRON_SWORD 1
  - GOLDEN_APPLE 2

```
